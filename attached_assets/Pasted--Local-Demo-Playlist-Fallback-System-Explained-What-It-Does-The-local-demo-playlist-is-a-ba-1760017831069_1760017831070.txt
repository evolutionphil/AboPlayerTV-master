üéØ What It Does
The local demo playlist is a backup/offline mode that automatically loads when:

‚ùå User's IPTV playlist URL fails to load
‚ùå Network connection is unavailable
‚ùå Server is down or unreachable
Instead of showing an error and blocking the user, the app falls back to a pre-packaged local M3U file (demoo.m3u) with sample content so users can still navigate and see how the app works.

   ‚Üì

   ‚Üì
8. USER SEES DEMO CONTENT

   - Movies work

// Triggered when network fails
fallbackToLocalDemo() {
    // 1. Switch to local file
    settings.playlist_type = "type1";
    settings.playlist_url = "demoo.m3u";  // ‚Üê LOCAL FILE
    
    // 2. Clear old cached data
    localStorage.removeItem('api_data');
    
    // 3. Hide error dialogs
    $('#network-issue-container').hide();
    
    // 4. Load the local demo playlist
    this.proceed_login();
    
    // 5. Notify user
    showToast("Demo Mode", "Using demo content");
}
